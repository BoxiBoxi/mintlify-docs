AI images help each scene communicate visually. Generate images after the script and scene plan are stable.

## Generate an image

1. Open a scene.
2. Review the visual prompt.
3. Click **Generate image**.
4. Preview the result.
5. Regenerate or edit the prompt if needed.

## Better prompts

Describe the subject, style, setting, and mood. Avoid vague prompts such as "make it better".

```text
A clean product workspace, vertical 9:16 composition, modern SaaS dashboard on laptop, bright natural light.
```

## Save credits

Edit the prompt before regenerating. Repeated image generation can use credits quickly.
