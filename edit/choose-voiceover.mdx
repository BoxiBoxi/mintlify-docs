Voiceover shapes the tone of the final video. Choose a voice that fits the audience, topic, and platform.

## Choose a voice

1. Open the project.
2. Go to **Voiceover**.
3. Preview available voices.
4. Select a voice.
5. Save the project.

## Tips

- Use a clear voice for educational videos.
- Use a warmer voice for storytelling.
- Keep the same voice across a series for consistency.

## Before rendering

Read the script aloud once. If it feels awkward, edit the script before generating audio.
