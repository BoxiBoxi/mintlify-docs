---
title: "Facebook Reel"
description: "Connect Facebook pages so YupVid can publish Facebook Reels."
---

**Facebook Reel** lets YupVid publish vertical videos to Facebook pages.

YupVid publishes to pages, not personal Facebook profiles.

## What you need

- A Facebook account.
- Admin or manager access to the Facebook page.
- Permission to allow YupVid to publish videos to the selected page.

## Connect Facebook

1. Open **Platforms**.
2. Select **Facebook Reel**.
3. Click **Connect**.
4. Log in with Facebook if prompted.
5. Select the pages YupVid is allowed to publish to.
6. Confirm the permissions.

You can select one page or multiple pages. Any page managed by your Facebook account can be selected if Facebook allows the permission.

## Publish to Facebook

After Facebook is connected, the selected pages can appear in:

- **Publish to accounts** in **Bulk Create**.
- **Publish to accounts** in **Article to Video**.
- **Publish to platforms** in **Projects**.

When a video is ready, click **Upload** next to the Facebook page to publish it.

## Notes

- Make sure you select the correct page before uploading.
- Review the video before publishing it.
- If the page does not appear, reconnect Facebook and check your page permissions.
