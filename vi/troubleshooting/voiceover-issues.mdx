---
title: "Voiceover issues"
description: "Troubleshoot YupVid voiceover generation."
---

Voiceover problems can come from script length, unsupported characters, or temporary generation errors.

## Fixes

- Shorten very long sentences.
- Remove unusual symbols.
- Check that the selected voice is available.
- Save the script and try again.

## Quality issues

If the voice sounds unnatural, rewrite the script in a more conversational style. Shorter sentences usually sound better.

## Render again

After editing the script or voice, preview the project before starting another render.
