---
title: "Credits and usage"
description: "Understand how YupVid credits are used."
---

YupVid uses credits for AI actions and rendering. This helps you control cost before creating the final video.

## Common credit actions

| Action | Typical usage |
| --- | --- |
| Script preview | Generates an editable script and scene plan |
| AI image | Creates or regenerates visual assets |
| Audio and video render | Produces voiceover and the final MP4 |

## Save credits

- Edit the script before rendering.
- Preview scenes before generating many images.
- Reuse a project style when creating similar videos.
- Render only after the voice, subtitles, and layout are final.

## Check usage

Open your account area and review **Credit history** to see which actions used credits.
