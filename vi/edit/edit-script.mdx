description: "Refine the voiceover script before rendering."
---

The script controls the voiceover, pacing, and subtitle text. Edit it before you spend credits on rendering.

## What to check

- The hook is clear in the first few seconds.
- Each sentence sounds natural when spoken.
- Claims are accurate.
- The call to action matches the video goal.

## Steps

1. Open the project.
2. Go to **Script**.
3. Edit the text.
4. Save changes.
5. Preview the updated scene plan.

## Writing tips

- Use short sentences.
- Replace jargon with plain language.
- Keep one idea per scene.
