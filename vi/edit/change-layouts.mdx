Layouts control how text, images, and scene elements appear in the 9:16 video frame.

## Change a layout

1. Open the project.
2. Select a scene.
3. Go to **Layout**.
4. Choose a layout.
5. Preview the scene.

## Layout tips

- Use simple layouts for educational content.
- Use image-focused layouts when the visual is the main message.
- Keep subtitles readable on mobile.
- Avoid putting important text near platform buttons.

## Consistency

Use a consistent layout style across videos in the same series.
