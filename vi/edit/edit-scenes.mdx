---
title: "Edit scenes"
description: "Adjust scene text, order, and visual direction."
---

Scenes split your script into visual moments. Good scenes make the final video easier to follow.

## Edit a scene

1. Open the project.
2. Select a scene.
3. Change the scene text or visual direction.
4. Save the scene.
5. Preview the sequence.

## Scene guidelines

- Use one message per scene.
- Keep scene text short enough for subtitles.
- Move important scenes earlier.
- Remove scenes that repeat the same point.

## Reorder scenes

Drag scenes into the order you want. Review transitions after moving scenes.
