---
title: "Video render issues"
description: "Fix video rendering problems."
---

Video render issues can happen when the project has missing assets, long scenes, or interrupted generation.

## Checklist

- Preview the project.
- Check for missing images.
- Confirm voiceover settings.
- Shorten long scenes.
- Avoid editing while a render is active.

## Retry safely

Fix the issue first, then start a new render. Repeating the same render without changes may fail again.

## Batch renders

For bulk projects, render a smaller test batch before queueing all videos.
