---
title: "Import issues"
description: "Fix problems importing URLs, articles, or documents."
---

Import issues usually happen when the source cannot be read cleanly.

## URL import

- Check that the URL is public.
- Remove tracking parameters.
- Try the canonical article URL.
- Paste the text manually if the site blocks extraction.

## Document import

- Use files with selectable text.
- Avoid scanned images unless OCR is supported.
- Split large documents into smaller files.

## Content cleanup

Remove navigation menus, ads, author bios, and unrelated footer text before generating the script.
