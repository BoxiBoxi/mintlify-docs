---
title: "Image generation issues"
description: "Fix AI image generation problems."
---

AI image issues often come from unclear prompts or missing scene context.

## Fix prompts

- Name the subject clearly.
- Add the setting.
- Specify style and mood.
- Keep the prompt focused on one image.

## Regenerate carefully

Regeneration can use credits. Edit the prompt before generating again.

## If an image fails

Try a simpler prompt and remove conflicting instructions.
