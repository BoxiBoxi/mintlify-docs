---
title: "Fix failed renders"
description: "Troubleshoot YupVid render failures."
---

A render can fail when source content, generated assets, or processing settings need attention.

## Quick fixes

1. Open the failed project.
2. Check the render error.
3. Preview the project.
4. Fix the reported issue.
5. Render again.

## Common causes

- The script is too long.
- A scene has missing visual content.
- A generated image did not complete.
- Voiceover generation failed.
- The project was edited during rendering.

## If it still fails

Duplicate the project, shorten the script, regenerate missing assets, and render again.
