---
title: "Render status"
description: "Track audio and video render progress."
---

Render status shows where the video is in the production queue.

## Common statuses

| Status | Meaning |
| --- | --- |
| Queued | The render is waiting to start |
| Rendering audio | YupVid is generating voiceover |
| Rendering video | YupVid is creating the MP4 |
| Complete | The file is ready to download |
| Failed | The render could not finish |

## Refresh status

Open the project or render queue to see the latest status. Large batches can take longer than single videos.

## Failed status

Open the failed render and check the error details. Then follow the failed render guide.
