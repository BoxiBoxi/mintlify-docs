After rendering finishes, YupVid provides a downloadable MP4 file.

## Download steps

1. Open the rendered project.
2. Click **Download MP4**.
3. Save the file.
4. Upload it to your publishing platform.

## File tips

- Rename files before uploading if you manage many videos.
- Keep a copy of the project link with your content calendar.
- Watch the downloaded MP4 once before publishing.

## If download fails

Refresh the project and try again. If the render is no longer marked **Complete**, wait for the status to update.
