---
title: "Bulk Create"
description: "Create multiple YupVid videos from scripts, media, and shared video settings."
---

**Bulk Create** helps you create scheduled video batches. Use it when you want to generate several videos without repeating the same setup for each one.

In YupVid, the Bulk Create screen includes script blocks, per-script import and media controls, publish scheduling, connected account selection, and shared video settings.

## When to use Bulk Create

Use **Bulk Create** when:

- You want to create up to 5 videos at one time.
- You already have several scripts ready.
- You want to schedule videos for publishing.
- You want to publish finished videos to connected accounts.
- You want each video to use the same language, voice, theme, watermark, and ending settings.

## Screen overview

The Bulk Create screen has three main areas:

- The script area in the center, where each video has its own script block.
- The publishing row under each script, where you choose **Publish time** and **Publish to accounts**.
- The right setup panel, where you configure **Video setup**, **Watermark**, and **Ending**.

At the top of the page, you can also use **Upload TXT**, **Add script**, and **Gen video**.

## Add scripts

Each script block represents one video.

You can add scripts in three ways:

- Type or paste text directly into **Script 1**.
- Click **Add script** to create another script block.
- Click **Upload TXT** to upload prepared text content.

Each script should contain one complete video idea. Keep the script clear and focused so YupVid can create better scenes.

## Import content for one script

Inside each script block, click **Import** if you do not want to type the script manually.

Use **Import** to upload a source file or bring in content from a URL, similar to the Studio workflow.

## Add media for one script

Click **Media** inside a script block to add images for that video.

You can use media to:

- Upload images from your computer.
- Search stock images.
- Generate images with AI.

Media is managed per script, so each video in the batch can have its own images.

## Remove a script

Click the delete icon on a script block if you no longer want to create that video.

Before deleting, make sure you do not need the script or its selected media.

## Publish time

Use **Publish time** to choose when the video should be published.

This is useful when you are preparing a batch in advance and want each video to go out on a schedule.

## Publish to accounts

Use **Publish to accounts** to select the connected accounts or pages where YupVid should publish the finished video.

The row can show accounts from platforms such as:

- Facebook Reel
- YouTube Short
- TikTok
- Threads
- Instagram
- LinkedIn
- X (Twitter)

If the account is connected correctly, YupVid can use the selected accounts to publish the video after it is created.

## Video setup

The **Video setup** panel applies shared settings to the batch.

You can configure:

- **Language**: The language of the video.
- **Video length**: **Short**, **Medium**, or **Full**.
- **Voice**: The voice used for narration.
- **Effect**: Overlay and transition effects.
- **Theme**: The visual style and color layout.

## Watermark

Use **Watermark** to show a brand or copyright text on the video.

You can configure:

- **Enable watermark**: Turn the watermark on or off.
- **Text**: The watermark text, such as `yupvid.com`.
- **Position**: Where the watermark appears.
- **Color**: The watermark color.

## Ending

Use **Ending** to configure the final scene of the video.

You can add:

- **Logo**
- **Logo text**
- **Website**
- **TikTok**
- **Facebook**
- **Instagram**
- **X (Twitter)**
- **Voice Over / Subtitle**

The ending is useful for showing your brand, website, and final call to action.

## Generate the videos

After you finish the scripts, media, publishing settings, and video setup, click **Gen video**.

YupVid will create videos for the scripts in the batch.

## Recommended workflow

1. Open **Bulk Create**.
2. Add or upload your scripts.
3. Add media for each script if needed.
4. Choose **Publish time** for each video.
5. Select the connected accounts in **Publish to accounts**.
6. Configure **Language**, **Video length**, **Voice**, **Effect**, and **Theme**.
7. Configure **Watermark** and **Ending**.
8. Click **Gen video**.

## Notes before generating

- Review each script before generating the batch.
- Keep each script focused on one topic.
- Make sure the selected voice matches the video language.
- Check that the selected platform accounts are correct.
- Use AI images if you do not have media ready.
