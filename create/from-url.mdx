---
title: "Create from URL"
description: "Import a public URL and create a YupVid project from it."
---

Use **URL** when the source content is already published online.

## Requirements

The URL should be public and readable without a login. If the page blocks access, copy the article text and use **Create from article** instead.

## Steps

1. Click **Create project**.
2. Choose **URL**.
3. Paste the link.
4. Click **Import**.
5. Review the extracted content.
6. Generate the script preview.

## If import fails

Try these fixes:

- Check that the link opens in a normal browser window.
- Remove tracking parameters from the URL.
- Copy the article text manually.
- Use **Upload document** if the source is a PDF or document.
